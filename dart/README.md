# 🎯 Dart Microservice

This directory contains the **Dart** implementation of our standardized JSON logging API.

## 🚀 Quick Start
You can easily spin up this individual service using the global run script from the root repository:

```bash
# From the root of the repository
./run.sh dart
```

## 🛠️ Tech Stack
- **Language**: Dart
- **Package Manager**: Pub
- **Default Port**: `8014`

## 📁 Source Details
- **Main Entrypoint**: `bin/server.dart`
- **Configuration**: `pubspec.yaml`
