# 🍔 Node.js Microservice

This directory contains the **Node.js** implementation of our standardized JSON logging API, complete with dependency management. 

## 🚀 Quick Start
You can easily spin up this individual service using the global run script from the root repository:

```bash
# From the root of the repository
./run.sh node
```

## 🛠️ Tech Stack
- **Environment**: Node.js
- **Package Manager**: NPM
- **Default Port**: `8012`

## 📁 Source Details
- **Main Entrypoint**: `index.js`
- **Configuration**: `package.json`
