# 🐘 PHP Microservice

This directory contains the **PHP** implementation of our standardized JSON logging API using the native PHP built-in web server.

## 🚀 Quick Start
You can easily spin up this individual service using the global run script from the root repository:

```bash
# From the root of the repository
./run.sh php
```

## 🛠️ Tech Stack
- **Language**: PHP >= 8.0
- **Server**: PHP Built-in Web Server (`php -S`)
- **Default Port**: `8013`

## 📁 Source Details
- **Main Entrypoint**: `index.php`
- **Configuration**: `composer.json` (if applicable)
