HOMEWORK MICRO-SERVICE WEEK 1

Services and default ports:
- rust (Rust) : 8011
- php (PHP) : 8013
- dart (Dart) : 8014
- java (Java) : 8015
- go (Go) : 8016
- python (Python) : 8017
- javascript (JavaScript) : 8018

Run one service:
./run.sh <service>

Run all services (starts in background):
./run.sh all

Stop all services started by ./run.sh all (same terminal session):
kill 0
