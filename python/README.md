# 🐍 Python Microservice

This directory contains the **Python** implementation of our standardized JSON logging API using Python's native `http.server` module for maximum simplicity.

## 🚀 Quick Start
You can easily spin up this individual service using the global run script from the root repository:

```bash
# From the root of the repository
./run.sh python
```

## 🛠️ Tech Stack
- **Language**: Python 3
- **Runner**: `python3`
- **Default Port**: `8017`

## 📁 Source Details
- **Main Entrypoint**: `server.py`
